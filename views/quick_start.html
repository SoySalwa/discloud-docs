First things first

We install Discloud C++

For starters, you must have Discloud C++ installed in your operative system, if your don’t have installed, install it, using the next command if your operative system is a distribution Linux as Debian or Fedora. Also you can to try use it locally.

If your OS Fedora (.rpm) in the terminal:

sudo dnf install disloud-library

If your OS is based on Debian (.deb) in the terminal:

sudo apt install disloud-library

Note: If some of these isn’t your operative system, you must to compile the source code for yourself. In the future i wait is to able to facilitate this process for another operatives systems, if you’ve issues compiling the project, you can join to the discord server for help you with this.
